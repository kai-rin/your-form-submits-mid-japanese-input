# あなたのフォーム、日本語入力中に送信されていませんか？

日本語入力中にWebフォームが意図せず送信されてしまう問題を解説する静的サイトのリポジトリです。IME（Input Method Editor）で変換を確定するためにEnterキーを押すと、フォームが送信されてしまい、ユーザーのテキスト入力が中断されます。

## なぜこれが問題なのか

この問題により、日本語ユーザーはWebフォーム上で一文を打ち終える前に意図しない送信が繰り返し発生します。チャットUI、AI生成ツール、カスタマーサポートウィジェットなどで特に深刻で、サービスが日本語ユーザーの体験を軽視しているという印象を与えかねません。

この問題は日本語に限らず、中国語（Pinyin）や韓国語（Hangul）など、IMEを使用するすべての言語に影響します。

## このサイトの内容

- 問題の仕組みと発生原因の解説
- すべてのモダンブラウザで動作するコード修正例（`isComposing` チェック）
- フォーム送信方法の設計代替案
