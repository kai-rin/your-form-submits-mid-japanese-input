# 당신의 폼, 한국어 입력 중에 제출되고 있지 않나요?

이 저장소는 웹 폼에서 흔히 발생하는 문제를 설명하는 정적 사이트입니다. IME(입력기)로 한국어를 입력할 때, Enter 키를 눌러 조합을 확정하면 폼이 의도치 않게 제출되어 사용자의 텍스트 입력이 중단됩니다.

## 왜 이것이 문제인가

이 문제로 인해 한국어 사용자는 웹 폼에서 한 문장도 완성하지 못한 채 의도하지 않은 제출이 반복됩니다. 채팅 인터페이스, AI 생성 도구, 고객 지원 위젯 등에서 특히 심각하며, 해당 서비스가 한국어 사용자의 경험을 전혀 고려하지 않는다는 인상을 줄 수 있습니다.

이 문제는 한국어뿐만 아니라 일본어, 중국어 등 IME를 사용하는 모든 언어에 영향을 미칩니다.

## 사이트 내용

- 폼 제출 방식의 설계 대안
