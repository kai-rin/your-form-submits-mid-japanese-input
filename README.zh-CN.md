# 你的表单在中文输入过程中就提交了

本仓库包含一个静态网站，解释了 Web 表单中一个常见问题。当用户使用 IME（输入法编辑器）输入中文时，按下 Enter 键确认候选字词会导致表单意外提交，打断用户的文字输入。

## 为什么这是个问题

这个问题会导致中文用户在 Web 表单中连一句话都无法完整输入。在聊天界面、AI 生成工具、客服聊天组件等场景中尤为严重，让用户觉得该服务完全没有考虑中文用户的使用体验。

这个问题不仅影响中文用户，也同样影响日语和韩语等所有使用 IME 的语言。

## 本站内容

- 问题的原理及成因说明
- 适用于所有现代浏览器的代码修复示例（`isComposing` 检查）
- 表单提交方式的设计替代方案
